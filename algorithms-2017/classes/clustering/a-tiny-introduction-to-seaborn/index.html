A tiny introduction to Seaborn

This page is based on a Jupyter/IPython Notebook: download the original .ipynb

Installing Seaborn

Seaborn is another data visuaization library, like matplotlib but a little less insane. Install it by using pip3.

pip3 install seaborn

Using Seaborn

First let’s read in our data.

Histograms

Seaborn doesn’t plug into pandas, so you can’t just do .plot. You need to import seaborn separately, then feed it the dataframe and the data.
